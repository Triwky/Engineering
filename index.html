10/13/23 - This week we learned about aerospace engineering and how it comprises of space exploration and aeronautical engineering. The four forces on a flying object are thrust, drag, lift, and weight. Thrust allows the flying object to have force to be able to fly through. Drag is the force that drags the flying object against air molecules. Lift allows the flying object to stay airborne. Weight pulls the flying object down.
